# Python Sales Analysis Project

This project analyzes retail sales data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Sales analysis by region
- Profit comparison by category
- Top customers identification

## Outcome
Generated business insights through data visualization.
